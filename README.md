# 크몽 웹개발 전문가 출범 폴더

작성일: 2026-04-01
목적: 웹개발(반응형 템플릿 기반) 전문가 등록부터 상품 노출, 견적 응대, 작품 전시까지 한 번에 실행할 수 있도록 설계한다.

## 폴더 구성

- `docs/`
  - `01_전체_사업_설계.md`
  - `02_서비스_상품_설계.md`
  - `03_브랜딩_및_포트폴리오_가이드.md`
  - `04_마케팅_캠페인_실행표.md`
  - `05_크몽_즉시업로드_문안_세트.md`
  - `06_크몽_최종_붙여넣기_패키지.md`
  - `07_홍보_운영_완성본.md`
- `templates/`
  - `index.html` (템플릿 빠른 구경용)
  - `template-a-landing.html` (원페이지 랜딩 템플릿)
  - `template-b-business.html` (소상공인/회사 소개형)
  - `template-c-service-page.html` (서비스형/기능형 페이지)
  - `template-common.css` (공통 반응형 스타일)
- `checklist/`
  - `업로드_전_체크리스트.md`
  - `의뢰인_요구사항_폼.md`
- `sample_data/`
  - `샘플_카피_문구.md`

## 지금 바로 해야 할 순서

1. `docs/06_크몽_최종_붙여넣기_패키지.md`에서 프로필·상품·FAQ·응답 텍스트를 바로 복사
2. `docs/07_홍보_운영_완성본.md`에서 썸네일 캡션·상품명 변형·첫 24시간 응대문구 반영
3. `templates/`의 HTML/CSS를 기준으로 3개 샘플에 로고/색상/문구 반영
4. `checklist/업로드_전_체크리스트.md`로 최종 점검
5. 업로드 후 24시간 동안 응답속도(최초 답장 시간) 확인

> 핵심: 처음엔 **작동 가능한 샘플 + 신속한 응답 + 납기 약속 준수**가 가장 빨리 문의를 만듭니다.
