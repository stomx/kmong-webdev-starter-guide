# 템플릿 다채롭게 구성하기 가이드 (ABC 즉시 적용)

누구나 처음부터 어렵지 않게 고칠 수 있게, 템플릿 A/B/C를 **난이도별·목적별로 뽑아 쓰는 버전**을 정리했습니다.

---

## 기본 규칙 (초보자용)

1. 제목 1줄, 부제 1줄은 꼭 바꾸기
2. 버튼은 페이지당 1개만 기본 CTA + 1개 보조 CTA
3. 숫자/기간/금액은 본인 기준으로 즉시 교체
4. 각 섹션은 "문제 → 해결 → 증거 → 신청" 순서가 가장 안전
5. FAQ 2개만 있어도 시작엔 충분

---

## Template A (랜딩형) 다채로운 구성

### A-1) 심플형(짧은 페이지)
- 구성: `Hero + 3개 장점 + 문의폼`
- 추천 대상: 단일 상품, 즉시 문의 유도
- 복붙 기준값:
  - 헤더 서브카피: "지금 바로 시작할 수 있는 [서비스]"
  - 버튼: "지금 무료 진단 받기"

### A-2) 전환형(강조형)
- 구성: `Hero + 문제정의 + 혜택 4개 + FAQ 3개 + 문의`
- 추천 대상: 경쟁이 치열한 업종
- 포인트: 숫자(예: 기간, 절감량, 회수일)을 넣어 신뢰성 강화

### A-3) 리드수집형
- 구성: `Hero + 장점 + 체크리스트 + 신청폼 + 개인정보 처리 안내`
- 추천 대상: 무료 진단, 상담예약을 먼저 모으는 페이지
- 포인트: 폼 필드 최소 3개(이름/연락처/요청)로 끝내기

### A-4) 상단 고정형
- 구성: `Hero + 신뢰마커 3개 + 장점 + CTA + 후기`
- 추천 대상: PC/모바일 모두 방문 비중이 높은 경우
- 포인트: 상단 신뢰마커는 "납기/수정/응답속도"로 고정

### A-5) 이벤트형
- 구성: `Hero + 혜택 패키지 + 기간 카운트 + 신청마감 CTA`
- 추천 대상: 프로모션 기간 알림이 필요한 경우
- 포인트: 카운트/마감은 과장 없이 실제 기준일 기반

---

## Template B (기업 소개형) 다채로운 구성

### B-1) 신뢰형
- 구성: `브랜드 스토리 + 핵심서비스 + 운영프로세스 + 포트폴리오`
- 추천 대상: 전문성 보여주기가 핵심인 상담형 프로젝트
- 포인트: 회사 소개 문장 2줄 이하로 단순화

### B-2) 파트너형
- 구성: `문제 해결 경험 + 협업 방식 + 고객 혜택 + 문의`
- 추천 대상: B2B 성격의 작업 또는 장기 협업
- 포인트: "무엇을 더 했는지"보다 "무엇을 같이 해결했는지" 강조

### B-3) 포트폴리오형
- 구성: `대표 성과 3개 + 상세 사례 + 도구/기술 스택`
- 추천 대상: 이전 작업 근거가 있어 신뢰를 쌓는 경우
- 포인트: 성과 수치가 없으면 "운영 효율성 개선 포인트"로 대체

### B-4) 브랜딩형
- 구성: `핵심 가치 + 고객 후속지원 + 브랜드 톤라인 + 문의`
- 추천 대상: 브랜딩/CI를 같이 제시해야 하는 경우
- 포인트: 아이덴티티 문구를 과장 없이 2개만 고정

### B-5) 소개+가격형
- 구성: `서비스 설명 + 패키지표 + 계약후킹 + 상담`
- 추천 대상: 프로젝트 범위를 먼저 보여주고 견적 논의로 이동
- 포인트: 가격을 모르면 "최소 기준" 또는 "맞춤 견적" 처리

## Template C (서비스형) 다채로운 구성

### C-1) 가격형
- 구성: `서비스 항목 + 패키지 + 포함/제외 + FAQ`
- 추천 대상: 가격 비교 문의가 많은 시장
- 포인트: 제외 항목은 반드시 마지막에 고정 1칸 배치

### C-2) 기능형
- 구성: `문제별 기능 + 사용 시나리오 + 단계별 진행`
- 추천 대상: 기능 상세 설명이 필요한 페이지
- 포인트: "기능 하나당 2문장 이하" 규칙 준수

### C-3) 패키지형
- 구성: `START/GROWTH/PRO + 비교표 + 상담 CTA`
- 추천 대상: 선택지 제공을 통한 전환 유도
- 포인트: 시작가/상태별 기준을 명확화

### C-4) 커뮤니티형
- 구성: `문의 후기 + 진행 프로세스 + 책임 범위`
- 추천 대상: 기존 작업 신뢰가 필요한 장기 고객군
- 포인트: 후기 대신 "가상 사례"도 가능하나 명시 필요

### C-5) 운영형
- 구성: `런칭 일정 + 인수인계 가이드 + 유지보수 정책 + 문의`
- 추천 대상: 장기 납품/재수정이 반복되는 고객군
- 포인트: 유지보수 범위를 먼저 정하고 문구로 고정

## 즉시 적용용 복붙 블록 (A/B/C 공통)

- CTA 버튼 텍스트
  - 1순위: `의뢰 시작하기`
  - 2순위: `상담 신청`
  - 3순위: `무료 진단 받기`
- 타이틀 교체형
  - `문자 12~20자` 권장 (너무 길면 모바일에서 줄바꿈 난잡)
- 위험 문구 교체형
  - "원하는 디자인에 맞춰 조정"
  - "요구사항 변경 시 일정 재조정"

## 마무리 체크

1. 초보는 **A-1 + B-1 + C-1** 조합부터 시작
2. 자신감 있으면 A-2/B-4/C-2 추가
3. 실전 운영은 **FAQ 2개, 포함/제외 1개 문단, 응답속도 1개 문장**만 먼저 넣어도 충분히 시작 가능
