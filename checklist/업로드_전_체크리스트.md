# 업로드 전 체크리스트

- [ ] 전문가 프로필:
  - [ ] 직무명, 전문영역, 응답시간 기재
  - [ ] 대표 사례 수량(최소 3개) 기재
  - [ ] 세금계산서 가능 여부 표기

- [ ] 상품설명:
  - [ ] 가격(공개가)<br/>
  - [ ] 납기 범위 기재(일 단위)
  - [ ] 포함/제외사항 분리 표기
  - [ ] 추가요청 과금 규칙 기재

- [ ] 템플릿:
  - [ ] 모바일/PC 뷰 2개 이상 캡처
  - [ ] 폼 동작(연락처/문의) 확인
  - [ ] 외부링크(지도/버튼/이메일) 점검

- [ ] 응대체계:
  - [ ] 공통 응답 템플릿 2개 저장
  - [ ] 의뢰인 체크리스트 준비
  - [ ] 수정/유지보수 범위 문구 고정

- [ ] 운영:
  - [ ] 가격/문구 1회 검토
  - [ ] 출금/정산 계정 정합성 확인
  - [ ] 출시일 기준 알림 예약
