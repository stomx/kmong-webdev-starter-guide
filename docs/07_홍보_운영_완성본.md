# 크몽 업로드 후 즉시 운영용 완성본

## 1) 썸네일 캡션(3종)

### 템플릿 A (랜딩)
- 제목: **반응형 랜딩페이지 1페이지**
- 서브카피: **모바일 완전 대응, CTA 중심 구조**
- 설명: **브랜드 핵심 메시지 정렬로 문의 전환율을 높인 샘플**

### 템플릿 B (기업형)
- 제목: **소상공인 홈페이지 제작형**
- 서브카피: **메뉴 구조 + 문의경로 + 지도 연동**
- 설명: **사업 소개형 페이지를 빠르게 정돈한 실제 제작 예시**

### 템플릿 C (서비스형)
- 제목: **서비스 소개형 페이지**
- 서브카피: **기능·패키지·FAQ 한눈에 정리**
- 설명: **견적 협업에 필요한 핵심 정보만 정리한 구성 샘플**

> 썸네일 1장은 PC, 1장은 모바일로 각각 한 장씩 준비해 각각의 템플릿명 아래에 배치하면 조회률이 올라갑니다.

## 2) 상품 제목 변형 10개

### 기본형
1. 반응형 랜딩페이지 제작 (심플형)
2. 소상공인용 반응형 홈페이지 제작
3. 퍼블리싱 정리/수정 패키지

### 클릭 유도형
4. 문의가 늘어나는 반응형 랜딩페이지
5. 빠른 오픈형 랜딩 제작 (PC/모바일 완성)
6. 첫 페이지부터 상담으로 이어지는 기업형 홈페이지

### 가격 신뢰형
7. 랜딩페이지 제작 49만원 (기준형)
8. 홈페이지 제작 89만원 (표준형)
9. 퍼블리싱 긴급 정리 29만원

### 기능 강조형
10. 반응형/문의폼/지도 포함 홈페이지 제작

> 추천: 제목은 1개는 가격형, 1개는 목적형, 1개는 기능형으로 A/B 테스트해도 좋습니다.

## 3) 첫 24시간 응대 메시지 3개 (복붙용)

### 메시지 A: 빠른 첫회신(초기 인사)
안녕하세요. 빠르게 확인했습니다. 기본 방향은 2가지 중 하나로 잡으면 견적이 가장 정확해집니다.
- 페이지 수 기준(랜딩형/소개형/기능형)
- 기능 범위 기준(문의폼/지도 포함 여부)

요청해주시면 오늘 중으로 정확 견적과 일정으로 다시 드리겠습니다.

### 메시지 B: 범위 명확화형
의뢰 내용 잘 받았습니다.
아래 두 가지만 먼저 보내주시면 바로 산정 가능합니다.
1) 현재 사이트 여부  2) 원하시는 페이지 수  3) 필요 기능(문의/지도/예약/결제)
범위가 정해지면 금액과 납기를 패키지 기준으로 즉시 제안드리겠습니다.

### 메시지 C: 신뢰 전달형
확인 감사합니다. 작업은 범위 기반 계약으로 진행되어서 변경 비용이 투명합니다.
원하시면 기본+선택 항목으로 견적안을 나눠서 드릴 수 있어요.
- 기본: 반응형, 구조 정리, 전달 포인트 구성
- 선택: 기능 연동, 유지보수, 긴급 수정

## 4) 업로드 직후 24시간 실행표

- 업로드 후 1시간 안: 메시지 템플릿 테스트(자기 메시지 문구 맞춤)
- 3시간 안: 썸네일 2세트(PC/모바일) 업로드 완료
- 6시간 안: FAQ 문구 2개 추가(주로 '납기'와 '수정횟수')
- 12시간 안: 응답속도 로그(첫 응답 시간) 기록
- 24시간 안: 대기중/연락 없음 문의 1개당 리마인드 멘트 준비
