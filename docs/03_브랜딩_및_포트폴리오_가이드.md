# 03. 브랜딩/포트폴리오 가이드

## 1) 브랜딩 톤앤매너

- 톤: 전문적이되 딱딱하지 않게(결과 중심)
- 핵심 메시지: "빠르게, 깔끔하게, 유지보수까지" 
- 키워드: 반응형, 모바일 최적화, 빠른 전달, 수정 범위 명시

## 2) 포트폴리오 3개 필수 구성

1. **랜딩 변환형**
   - 문제: 기존 랜딩 전환 저조
   - 작업: 구조 리디자인 + CTA 개선
   - 결과: 섹션 수, 모바일 사용성

2. **기업 소개형**
   - 문제: 정보 정리 불안정
   - 작업: 메뉴 구조 재정비 + 콘텐츠 흐름 정렬
   - 결과: 가독성, 접속 편의성

3. **서비스/상품형**
   - 문제: 상품 설명이 길고 복잡함
   - 작업: 카테고리 카드형 구성 + CTA 분산 배치
   - 결과: 기능 구분 명확화

## 3) 썸네일 규칙

- 썸네일 1장당 1키 메시지
- 템플릿 미리보기 + 실제 반응형 스크린샷(모바일 1장 필수)
- 제목은 실제 고객 문제를 반영
  - 예) "로컬 카페 오픈을 위한 모바일 반응형 소개형 페이지 3일 완성"

## 4) 가격대별 신뢰 장치

- 가격 노출: BASIC 49만 / STANDARD 89만 / PREMIUM 150만
- 별도 문구: “범위 내 수정 1~3회 포함, 추가 요청은 옵션 견적"
- 세금계산서 발행 여부: 가능 표시
