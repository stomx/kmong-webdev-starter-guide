# 원격지에서 확인하는 방법

## 1) 지금 바로 열기(권장)

터미널에서 아래 명령을 실행하세요.

```bash
cd /Users/jaymon/.openclaw/workspace/Kmong-WebDev-Starter
bash run_preview.sh
```

브라우저에서 접속:
- `http://127.0.0.1:7860/viewer/`

## 2) 다른 기기(원격)에서 보기

같은 네트워크라면 아래로 접속:
- `http://<내PC_IP>:7860/viewer/`

예시
- `http://192.168.0.12:7860/viewer/`

## 3) 확인할 항목

- 한눈에 보기: `00_한눈에_보기_요약본.md`
- 업로드용 최종 문안: `docs/06_크몽_최종_붙여넣기_패키지.md`
- 홍보/운영 완성본: `docs/07_홍보_운영_완성본.md`
- 템플릿 파일: `templates/index.html` 및 3개 템플릿

## 4) 이 뷰어에서 바로 확인되지 않는 부분

서버에서 일부 .md 파일은 브라우저가 텍스트 형태로만 보여줄 수 있습니다. 중요한 부분은 이미
`00_한눈에_보기_요약본.md`에 요약되어 있어 바로 읽기 쉽게 정리돼 있습니다.

## 5) 추가로 필요하면

원하시면 제가 즉시 **웹용 렌더 뷰어(마크다운 자동 변환)** 버전으로 바꿔서,
링크 클릭 한 번으로 깔끔하게 보이게 만들어드리겠습니다.
