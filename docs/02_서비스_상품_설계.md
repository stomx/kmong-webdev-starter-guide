# 02. 서비스 상품 설계(크몽 업로드 초안)

## 상품명 템플릿

### [상품 1] 반응형 랜딩페이지 제작 - 심플형
- 태그: 웹사이트 제작, 반응형, 랜딩, 퍼블리싱
- 가격(공개): 49만 원
- 소요기간: 4~6일
- 포함사항:
  - 헤더/히어로/서비스/문의 섹션
  - 모바일 반응형
  - 기본 SEO 세팅(메타, 타이틀)
  - 1회 수정
- 제외사항:
  - 콘텐츠 제작(문구/이미지)
  - 도메인·호스팅 구매/연결
  - 추가 기능(회원가입/로그인/결제)

### [상품 2] 소상공인 홈페이지 제작 - 표준형
- 가격(공개): 89만 원
- 소요기간: 8~12일
- 포함사항:
  - 5~10페이지
  - 문의폼 + 지도 + 블로그/게시판 1개
  - 모바일/PC 반응형
  - 2회 수정 + 파일 전달
- 제외사항:
  - 고급 기능(예약/결제/API)

### [상품 3] 퍼블리싱 + 유지보수 패키지
- 가격(공개): 29만 원 / 1회 단건
- 소요기간: 2~3일(요청사항에 따라)
- 포함사항:
  - 기존 화면 보완/정리
  - HTML/CSS/JS 수정
  - 크로스브라우저 점검
  - 1회 QA 점검

## 의뢰 문의 자동응답 템플릿

> 안녕하세요. 의뢰 감사합니다. 아래 정보를 기준으로 정확 견적으로 안내드리겠습니다.
> 1) 홈페이지 목적
> 2) 원하시는 페이지 수
> 3) 필요 기능(문의폼/지도/예약/회원)
> 4) 기한
> 5) 도메인/브랜딩 자료

제공 후 24시간 내 맞춤 견적 제시.

## FAQ(클릭형 질문에 바로 붙일 답)

- 수정은 몇 회까지 되나요? → 상품별로 1~3회
- 유지보수는 가능한가요? → 유료 패키지 또는 별도 계약
- 모바일 반응형 가능한가요? → 전 범위 반응형 지원
- 결제/도메인은 어떻게 하나요? → 크몽 결제 후, 외부도메인 연동 필요 시 별도 안내
