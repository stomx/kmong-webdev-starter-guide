# 01. 전체 사업 설계

## 1) 포지션 (1개만 확정)

현재 시장 진입을 빠르게 하려면 가격과 범위를 먼저 고정해야 한다.

### 추천 초기 포지션(우선순위)

1. **반응형 원페이지/랜딩 제작 (1~1개월 단기)**
2. **중소형 기업형 소개형 홈페이지**
3. **퍼블리싱/마크업 유지보수 패키지**

> 이유: 난이도 대비 납기 관리가 쉽고, 샘플 제작 비용이 낮으며, 후기 모으기 빠름.

## 2) 타깃 고객 페르소나

- 1인/소규모 사업자 (카페, 스튜디오, 교육, 로컬서비스)
- 브랜드 리뉴얼이 필요한 카피/로컬 업체
- 쇼핑몰 리뉴얼/랜딩 개선이 필요한 초기 창업자

## 3) 제공 상품 3단계 모델

- **BASIC (입문형)**: 1 페이지~3페이지, 콘텐츠 최소 구성, 기본 반응형
- **STANDARD (주력형)**: 5~10페이지, 메뉴 구조 정리, 기본 문의 처리, SEO 기초
- **PREMIUM (고급형)**: 유지보수 포함(2주/1회), 간단 기능 연동(문의폼, 지도, SNS)

## 4) 가격(시작값 제안)

초기 공개가는 아래처럼 시작하고, 실제는 역량/케이스별로 ± 조정.

- BASIC: 49만 원
- STANDARD: 89만 원
- PREMIUM: 150만 원

> 절대 ‘최저가 전쟁’으로 들어가지 말 것. 기본 범위·제외 항목을 명확히 기재.

## 5) 운영 규칙

- 추가 요청은 항상 **별도 항목(부가비)**으로 전환
- 납기 약속: 기본 7~14일(협의 시)
- 수정 횟수: BASIC 1회, STANDARD 2회, PREMIUM 3회(최초 범위 내)

---

**출처 기반 반영 포인트**
- 카테고리 가격이 넓고, 유사 서비스 밀집도가 높음
- 평균가 기준으로는 중간가격대 + 명확한 범위가 전환율이 좋음
- 세금계산서/빠른 응답/포트폴리오 노출은 신뢰 신호로 작동
