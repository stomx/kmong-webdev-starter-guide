# 즉시 업로드 문안 세트 (복사/붙여넣기용)

아래 문안은 바로 크몽 판매 페이지에 넣을 수 있게 **문장 단위**로 정리한 것입니다.

---

## 1) 전문가 프로필 기본문

안녕하세요, 웹 퍼블리싱·반응형 웹 개발 전문으로 사업을 진행하는 크몽 전문가입니다.  
짧은 기간 안에 **확장성이 높은 웹 페이지**를 빠르게 제작하고, 의사소통이 명확한 진행 방식으로 결과물을 전달합니다.

### 경력/강점 한 줄 소개
- ① 반응형 웹 제작(모바일 최적화)  
- ② 퍼블리싱 정교화(페이지 구조/가독성/전환 경로)  
- ③ 의뢰인 요구사항 기반 범위 산정과 투명한 견적

### 응답/협업 규칙
- 영업일 기준 24시간 내 최초 답변
- 범위 기반 제작(요청사항 미기재 항목은 별도 산정)
- 수정 횟수, 납기, 제외 항목을 계약 전에 선확정

## 2) 상품별 업로드 문안

### 상품명: 반응형 랜딩페이지 제작 - 심플형
**가격:** 49만 원 (최초 공개가)

#### 요약 설명(한 줄)
"핵심 메시지 하나를 빠르게 전달하는 반응형 랜딩 페이지"

#### 상세 설명
- 페이지 구성: 히어로, 장점/특징, CTA, 문의 영역
- 지원 디바이스: PC/태블릿/모바일 반응형
- 기본 SEO 메타 구성(타이틀/설명)
- 포함: 수정 1회(범위 내)
- 기간: 4~6일(협의)

#### 포함 항목
- 구조 설계(메인 메시지 → 행동 유도)
- 반응형 퍼블리싱
- 기본 폼/버튼 동작 구성
- PC·모바일 테스트

#### 제외 항목
- 도메인/호스팅 구매 및 연동 설정(별도)
- 콘텐츠(카피/이미지/로고) 신규 제작
- 결제/회원/예약 등 고도 기능

#### FAQ에서 우선 노출할 문구
- 납기: 4~6일
- 수정: 1회
- 결제 후 작업 시작

### 상품명: 소상공인 홈페이지 제작 - 표준형
**가격:** 89만 원 (최초 공개가)

#### 요약 설명(한 줄)
"사업 목적이 분명한 5~10페이지 소개형 홈페이지"

#### 상세 설명
- 페이지: 최대 10페이지 내외(메뉴 구조 협의)
- 포함: 모바일·PC 반응형, 문의폼, 지도, 기본 아카이브/게시 영역 1개
- 포함: 수정 2회(범위 내)
- 기간: 8~12일(협의)

#### 포함 항목
- 전체 IA(메뉴 구조 정리)
- 섹션별 문구 가이드 반영
- 핵심 CTA 배치
- 기본 접근성 체크(텍스트 크기/대비/레이아웃)

#### 제외 항목
- 예약/결제 API, 대규모 연동 개발
- 도메인 구매·연결
- 반복 수정(범위 초과 항목)

**가격:** 29만 원 (1회 단건, 조정 가능)

- 테스트 링크/브라우저 점검

#### 제외 항목
- 신규 대규모 기능 개발
- 반복 긴급 요청(별도 협의)

## 4) 자주 묻는 질문(바로 붙여 넣기)

Q. 납기는 얼마나 걸리나요?  
A. 기본 4~12일입니다. 페이지 수/기능량에 따라 조정 가능합니다.

Q. 수정은 몇 번까지 가능한가요?  
A. 상품별로 1~2회(범위 내) 포함됩니다. 범위 초과는 사전 합의 후 진행합니다.

Q. 세금계산서 발행이 가능한가요?  
A. 가능합니다. 결제 전 신청자 정보를 기준으로 처리 절차를 안내드립니다.

Q. 추가 기능(예약/결제)은 가능한가요?  
A. 가능합니다. 기능 범위를 분리해 산정하면 견적이 명확합니다.

1) 현재 사이트 유무
2) 원하는 페이지 수
3) 필요 기능(문의폼/지도/예약/결제/예약)
4) 희망 오픈일
5) 참고 사이트 2개

지금 보내주시면 기준 단가 기준으로 바로 범위를 제안드리겠습니다.
