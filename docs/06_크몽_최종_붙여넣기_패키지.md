# 크몽 업로드용 최종 붙여넣기 패키지

아래 텍스트는 실사용용으로 바로 붙여넣기 가능합니다.

## 1) 전문가 소개(프로필 상단에 사용)

안녕하세요. 웹 퍼블리싱·반응형 홈페이지 제작을 맡아 운영하고 있는 크몽 전문가입니다.

핵심은 단순히 예쁜 화면이 아니라, **빠르게 오픈 가능한 실사용 페이지**를 만드는 것입니다.

- 반응형 퍼블리싱(PC·태블릿·모바일)
- 의뢰인 맞춤 요구사항 기반 산정
- 계약 전 포함/제외 항목을 명확히 고지

응답은 영업일 기준 24시간 이내, 범위와 일정은 계약 전에 확정 후 진행합니다.

---

## 2) 상품명/설명 복붙본

### 상품 1

**반응형 랜딩페이지 제작 - 심플형**

핵심 메시지를 하나로 정리해 빠르게 문의가 일어나는 랜딩 페이지를 제작합니다.

### 상품 요약
- 가격: 49만원 (최초 공개가)
- 소요기간: 4~6일
- 포함: 반응형 퍼블리싱, 기본 문의폼, CTA 설계, SEO 기본 메타
- 수정: 1회(범위 내)

### 포함 항목
- 히어로 섹션 설계(메인 메시지)
- 섹션형 콘텐츠 구조(문제/해결/신뢰/문의)
- 모바일 반응형 최적화
- 기본 브라우저 테스트(크롬 기준)

### 제외 항목
- 도메인·호스팅 구매/연결
- 이미지/카피 전체 제작
- 결제/예약/회원 기능

문의가 오면 아래 기준으로 24시간 내 정확 견적 안내

---

### 상품 2

**소상공인 홈페이지 제작 - 표준형**

브랜드/서비스를 소개하고 문의 전환이 쉬운 기업형 홈페이지를 제작합니다.

### 상품 요약
- 가격: 89만원 (최초 공개가)
- 소요기간: 8~12일
- 포함: 5~10페이지, 반응형, 문의폼, 지도, 기본 게시 영역 1개
- 수정: 2회(범위 내)

### 포함 항목
- 메뉴 구조/페이지 구성(IA 정리)
- 문구/섹션 배치 가이드 반영
- 모바일/PC 레이아웃 일치도 검수
- 결과물 전달 및 검수 대응

### 제외 항목
- 예약/결제/고급 API 연동
- 도메인 구매/연동은 별도
- 반복 범위 확장 수정

---

### 상품 3

**퍼블리싱 수정 + 단기 유지보수 패키지**

기존 사이트의 깨진 레이아웃, 반응형 오류, 퍼블리싱 개선을 빠르게 정리합니다.

### 상품 요약
- 가격: 29만원 (1회 단건)
- 소요기간: 2~3일
- 포함: HTML/CSS/JS 정돈, 반응형 수정, 크로스브라우저 점검
- 수정: 1회(범위 내)

### 포함 항목
- 화면 깨짐/간격/정렬 이슈 보정
- 기본 스크롤/버튼 동작 점검
- 모바일/태블릿 레이아웃 확인

### 제외 항목
- 신규 기능 대규모 개발
- 2회 이상 반복 긴급 수정(별도 계약)

## 3) 문의 응답 템플릿(복붙)

안녕하세요. 문의 감사합니다.
정확한 견적을 위해 아래 내용을 보내주시면 24시간 내 답변드리겠습니다.

1) 웹사이트 현황(없음/보유 중)
2) 희망 페이지 수
3) 필요한 기능(문의폼/지도/예약/결제)
4) 목표 오픈일
5) 예산 범위
6) 참고 URL 2~3개

주신 내용으로 구성 범위를 정의해 범위·가격·일정을 같이 제안드리겠습니다.

## 4) 자주 묻는 질문(바로 붙여넣기)

**Q. 납기는 어느 정도인가요?**
A. 기본 4~12일입니다. 난이도에 따라 변동될 수 있어요.

**Q. 수정은 몇 번까지 되나요?**
A. 상품별로 1~2회 포함되고, 추가 요청은 사전 협의 후 진행됩니다.

**Q. 세금계산서 발급이 가능한가요?**
A. 가능합니다. 결제 후 요청하시면 절차를 안내드립니다.

**Q. 도메인/호스팅은 누가 처리하나요?**
A. 구매/연결은 별도 항목으로 안내드리며, 필요 시 연동 가이드를 드립니다.
